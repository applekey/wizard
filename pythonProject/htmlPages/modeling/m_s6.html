In Step 6 you will generate the socket from the now rectified positive model. Start by using the selection tool to trace the trim line for the socket as marked during the sca nning process, Make sure to trace above the line rather than directly on the line and verify that t he line is continuous and that there are no gaps. When ready to proceed, press To generate a socket, double-click anywhere below the trim line to select the appropriate part of the model. Then, press

to create the socket. You can set the space between the residual limb and the socket using the ‘inner dimension’ slider and the thickness of the socket wall using the ‘outer dimension’ slider.
